The Pedagogical Mission: SGD Intuition
Traditional AI education often leads to "Tutorial Hell"—students can write the code for Stochastic Gradient Descent (SGD) but lack the physical intuition of how learning rates impact convergence.
Linear Ascent bridges this gap using Arithmetic Progression (AP):
 * The Origin (a): Represents the Initial Bias/Weight.
 * The Pulse (d): Represents the Learning Rate (\eta).
 * The Target: Represents the Global Minimum in a loss landscape.
 
Technical Mapping
| Game Element | ML Concept | Technical Implication |
|---|---|---|
| Common Difference (d) | Learning Rate | Determines the step size in weight space. |
| Overshooting | Exploding Gradient | High learning rate causes the model to jump over the target. |
| Stagnation | Vanishing Gradient | Low learning rate leads to failure to converge within steps. |
